
**问题 2**: 日期对象引用
```javascript
// 修复前
const end = new Date(date)  // 引用同一对象
end.setDate(date.getDate() + 6)  // 影响原对象

// 修复后
const end = new Date(date.getTime())  // 克隆新对象
end.setDate(date.getDate() + 6)  // 不影响原对象
```
