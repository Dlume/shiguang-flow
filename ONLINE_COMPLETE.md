## 📞 需要帮助？

### 如果手机无法访问局域网

1. **检查防火墙**：
   ```bash
   # Windows 防火墙允许 8080 端口
   netsh advfirewall firewall add rule name="Node Server" dir=in action=allow protocol=TCP localport=8080
   ```

2. **检查 IP 地址**：
   ```bash
   ipconfig | findstr IPv4
   ```

3. **重启服务**：
   ```bash
   cd D:\QwenPawOut001\projects\ShiGuang-Flow\dist
   npx serve . -l 8080
   ```
